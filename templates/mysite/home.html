Home

You are viewing the public homepage of the ZITADEL Django example web app.

Public pages

Browsing to one of the following pages will NOT trigger OIDC authentication flow.

  • user: View profile data of the authenticated Django-user, if not authenticated then option for login. No special roles required.
  • login: Login to AuthenticationBackend.
  • polls: List all availbe polls. No special roles required. Login required to see details of polls and to vote, Staff role required to see results.

Authenticated pages

Browsing to one of the following pages will trigger OIDC authentication flow.

  • admin: User management by Django. Admin role required.
